Linux:(root)

In terminal(2): python client.py

Windows: (root/WindowsVersion)
Open IDLE and load server.py
press F5
Open IDLE and load client.py
press F5

In client shell send a message
In server terminal receive message, send response

Reason for two different versions:

Windows: Input
Linux/Unix:  Raw_Input
